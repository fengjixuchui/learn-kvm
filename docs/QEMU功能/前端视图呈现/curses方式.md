# curses方式

 * -display curses == -curses
 * 通过 curses 接口显示文本模式的输出。如果图形设备处于图形模式或者图形设备不支持文本模式则什么也不输出
 * 这种现实方式直接将当前终于当成虚拟机的终端。**可以用快捷键 Esc+1,2 在监视器终端和虚拟机终端来回切换。**
